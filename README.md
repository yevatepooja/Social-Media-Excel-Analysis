# Social-Media-Excel-Analysis
## Project Overview
A data analysis project built using Microsoft Excel to analyze social media campaign performance and user engagement.

## Objectives
- Clean and preprocess social media data.
- Analyze engagement metrics.
- Compare platform performance.
- Study hashtag effectiveness.
- Evaluate campaign performance.
- Analyze follower retention and loyalty.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Conditional Formatting
- Data Cleaning
- Data Visualization

## Project Files
- Task 1 – Data Preprocessing and Cleaning
- Task 2 – Engagement Analysis
- Task 3 – Platform Performance Analysis
- Task 4 – Hashtag and Content Strategy
- Task 5 – Campaign Effectiveness
- Task 6 – Follower Retention & Loyalty

## Key Insights
• Identified the platform with the highest engagement.
• Compared campaign effectiveness across platforms.
• Evaluated the relationship between ad spend and follower growth.
• Analyzed content performance using hashtags.
• Studied follower retention trends.
