# 📊 Social Media Excel Analysis Project

## 📌 Project Overview

This project demonstrates data analysis and visualization using **Microsoft Excel**. The dataset was analyzed to evaluate social media performance, user engagement, campaign effectiveness, hashtag strategy, and follower retention through dashboards, pivot tables, charts, and data visualization techniques.

This project showcases practical Excel skills used in business analytics and marketing performance analysis.

---

## 🎯 Project Objectives

- Clean and preprocess raw social media data.
- Analyze engagement metrics across platforms.
- Compare platform performance.
- Evaluate hashtag and content strategy.
- Measure campaign effectiveness.
- Analyze follower retention and loyalty.
- Create interactive dashboards and charts.

---

- Data Cleaning

- Sorting & Filtering

- Dashboard Creation
